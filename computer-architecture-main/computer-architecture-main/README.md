## Repository Overview

This repository contains 10 laboratory exercises for the Computer Architecture course. The work focuses on VHDL-based digital design, simulation, and fundamental CPU architecture concepts.

Each lab is organized in a separate folder (Lab1 to Lab10) and follows a consistent structure for implementation and documentation.

---

## Lab Structure

Each lab folder contains:

- README.md — Lab description and documentation
- design.vhd — VHDL design implementation
- testbench.vhd — Testbench for simulation
- simulation.vcd — Simulation output file
- output.png — Simulation result screenshot
